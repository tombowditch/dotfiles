## VIM cheatsheet

### telescope

* ;f - find files
* ;F - main file browser
* ;g - grep
* \\ - buffers
* ;t - tags
* ;; - go back to what you just had open in TS
* ;e - diagnostics

### lsp
* K - hover
* gd - view refs/defs
* gR - rename
